# Excel-DashBoard-Project
Excel dashboard for churned customer analysis

### Overview
This project showcases an interactive Excel dashboard built to analyze sales performance across regions, products, and time periods.

### Features
- Interactive KPIs and charts
- Sales trend analysis
- Top-performing products and regions
- Clean and user-friendly dashboard layout

### Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

### How to Use
1. Download the Excel file
2. Enable editing and content
3. Use slicers to interact with the dashboard
